---

## Project Overview

This project automates performance testing of a sample API workflow using Apache JMeter integrated with Jenkins. On every build trigger, Jenkins checks out the latest test scripts from GitHub, executes the JMeter test in non-GUI mode, generates a full HTML performance dashboard, and archives results for historical comparison.

**Testing scope:**
- End-to-end API performance load testing
- Response time, error rate, and percentile analysis
- Automated report generation and CI/CD publishing

## How to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/praveenkumarbharthi/xyz.git
   ```

2. Run the JMeter test from the project root:
   ```bash
   scripts\runJmeter.bat
   ```

3. Open the JMeter HTML report:
   ```bash
   reports\html\index.html
   ```

4. (Optional) Generate the AI performance dashboard:
   - Install dependencies:
     ```bash
     pip install -r requirements.txt
     ```
   - Set the Gemini API key and run the analyzer:
     ```powershell
     $env:GEMINI_API_KEY = "YOUR_API_KEY"
     python .\scripts\ai_analyze.py
     ```
   - Open the AI dashboard:
     ```bash
     reports\ai\index.html
     ```

## Jenkins Pipeline Flow

```
Checkout Source
     ↓
Verify Environment
     ↓
Run JMeter Test
     ↓
Validate Thresholds
     ↓
🤖 AI Performance Analysis
     ↓
Publish HTML Reports
```

### Jenkins Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| ENVIRONMENT | dev | Target environment (dev / qa / stage) |
| THREAD_COUNT | 10 | Number of concurrent virtual users |
| LOOP_COUNT | 5 | Iterations per thread |
| RAMP_UP | 10 | Ramp-up period in seconds |

### Performance Thresholds

| Metric | Threshold |
|--------|-----------|
| Error Rate | < 1% |
| Average Response Time | < 5000 ms |

## AI Performance Dashboard

The repository now includes an AI-powered dashboard generated by `scripts/ai_analyze.py`, which converts JMeter results into a modern, dashboard-style HTML report.

The AI dashboard includes:
- polished KPI cards for build status, request volume, error rate, and response time
- progress bar-based health scoring and letter grade
- top slow endpoint bar chart
- color-coded endpoint health badges
- responsive dark-theme layout with dashboard sections
- AI-driven analysis and recommendations from Google Gemini

The AI report is generated at:
- `reports/ai/index.html`
